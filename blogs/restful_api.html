RESTFUL API

RESTFUL API là 1 tiêu chuẩn để thiết kế API cho ưngs dụng để tiền quản lí resource

API Application Programming Interface là tập hợp cơ chế, quy tắc để ứng dụng hay thành phần tương tác với ứng dụng hay thành phần khác Kiểu dữ liệu trả về: XML hoặc JSON

REST REpresentational State Transfer dạng chuyển đổi cấu trúc dữ liệu, kiểu kiến trúc để viết API. sử dụng phương thức HTTP để giao tiếp giữa các máy. REST gửi 1 yêu cầu GET, POST, PUT, DELETE đến 1 url để xử lý RESTFUL API k sử dụng session hay cookie để authenticate mà dùng 1 access token với mỗi request Khi request API sẽ có status code trả về
